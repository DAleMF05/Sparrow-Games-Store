# Sparrow Games Store

## Temática elegida
Tienda de videojuegos donde el usuario puede navegar un catálogo paginado (juegos manuales + juegos consumidos desde MockAPI), agregar juegos al carrito, y dar de alta nuevos juegos mediante un formulario reactivo.
